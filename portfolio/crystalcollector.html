Crystal Collector

An interactive web browsers game that dynamically updates the HTML with the jQuery library.

The player will be given a random number at the start of the game. There are four crystals displayed as buttons on the page and the value of each crystal is hidden until clicked. The player wins the game by matching the total score to random number. The game restarts whenever the player wins or loses and the value of each crystal will change.
